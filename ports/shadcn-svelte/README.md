# Startino Themes — shadcn-svelte port

Drop-in shadcn-svelte theme tokens for Startino's four palette flavors. Replaces the `:root` block in your project's `app.css`.

## What this is

Each file exports the full set of CSS custom properties that shadcn-svelte components read (e.g. `--background`, `--primary`, `--ring`, `--sidebar-*`). The values use modern `oklch(...)` notation, consistent with Tailwind v4 and shadcn-svelte's current conventions. Pick one flavor and drop it in — no JavaScript, no build step.

Available flavors:

| File | Flavor | Mode |
|------|--------|------|
| `dist/startino-mercury.css` | Mercury | Light |
| `dist/startino-mars.css` | Mars | Dark |
| `dist/startino-jupiter.css` | Jupiter | Dark |
| `dist/startino-neptune.css` | Neptune | Dark |

## Install (Tailwind v4)

**Option A — copy the file into your project:**

```css
/* app.css */
@import "./startino-neptune.css";
```

**Option B — reference by path from this repo** (no published npm package yet; consumers pull by path):

```css
/* app.css */
@import "@startino/themes/shadcn-svelte/dist/startino-neptune.css";
```

This replaces the `:root { ... }` block that `npx shadcn-svelte init` generates. Remove or comment out the existing block before importing.

## Switch flavors

Replace the import with another flavor file. There is no runtime flavor-switching built in — each file is self-contained and sets `:root` directly.

```css
/* swap Neptune for Mercury (light mode): */
@import "./startino-mercury.css";
```

## Customisation

Each variable maps to a Catppuccin-equivalent palette role. If you want to change which role drives a given variable, edit your local copy of the file. The mapping is:

| shadcn variable | Startino role |
|----------------|---------------|
| `--background` | `base` |
| `--foreground` | `text` |
| `--card` | `mantle` |
| `--card-foreground` | `text` |
| `--popover` | `surface0` |
| `--popover-foreground` | `text` |
| `--primary` | `green` (mint) |
| `--primary-foreground` | `crust` |
| `--secondary` | `pink` |
| `--secondary-foreground` | `crust` |
| `--muted` | `surface1` |
| `--muted-foreground` | `subtext0` |
| `--accent` | `surface2` |
| `--accent-foreground` | `text` |
| `--destructive` | `red` |
| `--destructive-foreground` | `crust` |
| `--border` | `overlay0` |
| `--input` | `surface0` |
| `--ring` | `green` |
| `--chart-1..5` | green, pink, blue, peach, mauve |
| `--sidebar-*` | mirrors main roles above |
| `--radius` | `0.625rem` (literal) |

All files are generated by `prism` — do not edit the `dist/` files by hand. To regenerate:

```sh
cd /shared/themes
bun compiler/prism.ts --port shadcn-svelte
```
