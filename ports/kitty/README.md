# Startino Themes — Kitty

## What this is

Kitty terminal color themes for the four Startino flavors: Mercury (light), Mars, Jupiter, and Neptune (dark). Generated by `prism` from the Startino palette; do not edit the `.conf` files by hand.

## Install

Copy the flavor you want into Kitty's themes directory:

```sh
cp dist/startino-neptune.conf ~/.config/kitty/themes/
```

Then reference it from your main config:

```
# ~/.config/kitty/kitty.conf
include themes/startino-neptune.conf
```

Restart Kitty (or send `SIGUSR1` for a live reload on supported platforms).

## Switch flavors

Change the `include` line to point at a different file:

| Flavor   | File                          | Style       |
|----------|-------------------------------|-------------|
| Mercury  | `startino-mercury.conf`       | Light       |
| Mars     | `startino-mars.conf`          | Medium-dark |
| Jupiter  | `startino-jupiter.conf`       | Dark        |
| Neptune  | `startino-neptune.conf`       | Darkest     |

```
include themes/startino-neptune.conf
```
