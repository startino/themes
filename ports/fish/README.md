# Startino Themes — fish

Fish shell color themes matching the Startino palette. Compatible with `fish_config theme choose` (fish 3.4+).

## What this is

Four dark-first themes (Mercury, Mars, Jupiter, Neptune) plus one light theme (Mercury) that map the Startino brand palette onto fish's color role system. Colors are assigned by role — green is used for quotes, options, and remote-host prompts (our primary accent); pink for redirections, operators, and pager prefix; and so on through the full Startino palette.

## Install

Copy the theme files to your fish themes directory:

```fish
cp "dist/Startino Neptune.theme" ~/.config/fish/themes/
cp "dist/Startino Jupiter.theme" ~/.config/fish/themes/
cp "dist/Startino Mars.theme" ~/.config/fish/themes/
cp "dist/Startino Mercury.theme" ~/.config/fish/themes/
```

Then activate your preferred flavor:

```fish
fish_config theme choose "Startino Neptune"
```

The change takes effect immediately in all open fish sessions.

## Switch flavors

| Flavor | Character | Style |
|---|---|---|
| Neptune | Dark, deep blue-green base | Default recommendation |
| Jupiter | Dark, warm mauve base | Slightly warmer dark |
| Mars | Dark, muted warm tones | Low-contrast dark |
| Mercury | Light | High-contrast light |

Switch at any time:

```fish
fish_config theme choose "Startino Mercury"
fish_config theme choose "Startino Mars"
fish_config theme choose "Startino Jupiter"
fish_config theme choose "Startino Neptune"
```

## Bundled in fish 4.4+ note

Catppuccin's fish port was bundled into the official fish 4.4+ release and is available without any manual install step. Startino's port has no such status — manual install (copying the `.theme` files) is the only path.

## Generated

These files are generated by the prism compiler from `fish.njk`. Do not edit `.theme` files by hand — regenerate with:

```sh
cd /shared/themes && bun compiler/prism.ts --port fish
```
