# Startino Themes — Waybar

GTK `@define-color` palette for [Waybar](https://github.com/Alexays/Waybar), the status bar for Hyprland and other wlroots compositors.

## What this is

A set of CSS files — one per Startino flavor — that declare all 26 color roles as GTK named colors using `@define-color`. Once imported, you can reference any role by name (`@base`, `@text`, `@green`, `@pink`, etc.) anywhere in your waybar `style.css`.

Brand aliases are also defined:
- `@accent`, `@primary` → mint green (the Startino primary)
- `@secondary` → pink

## Install

Copy the flavor file you want to `~/.config/waybar/themes/`:

```sh
cp startino-neptune.css ~/.config/waybar/themes/
```

Then import it at the top of your waybar `style.css`:

```css
@import url("themes/startino-neptune.css");
```

## Usage example

After importing, use any palette role as a GTK named color:

```css
window#waybar {
    background-color: @base;
    color: @text;
    border-bottom: 2px solid @surface0;
}

#workspaces button {
    color: @subtext0;
    background-color: @surface0;
}

#workspaces button.active {
    background-color: @green;
    color: @crust;
}

#workspaces button:hover {
    background-color: @surface2;
    color: @text;
}

#clock, #battery, #network, #pulseaudio {
    color: @text;
    background-color: @surface0;
    padding: 0 10px;
}

#battery.warning {
    color: @peach;
}

#battery.critical {
    color: @red;
}
```

## Switch flavors

Change the `@import` line to switch themes:

| Flavor   | File                       | Character |
|----------|----------------------------|-----------|
| Mercury  | `startino-mercury.css`     | Light     |
| Mars     | `startino-mars.css`        | Warm dark |
| Jupiter  | `startino-jupiter.css`     | Mid dark  |
| Neptune  | `startino-neptune.css`     | Deep dark |

```css
/* swap one line: */
@import url("themes/startino-jupiter.css");
```

## Generated files

These files are generated by [prism](../../compiler/prism.ts) from [`waybar.njk`](./waybar.njk). Do not edit by hand — regenerate with:

```sh
cd /shared/themes && bun compiler/prism.ts --port waybar
```
