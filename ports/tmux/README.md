# Startino Themes — tmux

## What this is

A tmux status-line theme that matches Startino's brand colors, available in 4 planet flavors: Mercury (light), Mars, Jupiter, and Neptune (dark). Generated by the prism compiler from `tmux.njk`.

Each `.conf` file exports color variables (`@thm_*`) for all palette slots plus a hardcoded `@thm_primary` (mint green) and `@thm_secondary` (pink), then applies them to the status bar, pane borders, window list, and mode styling.

## Install via TPM

> **Note:** This repo is not yet on TPM's registry. Manual install is the current path (see below). When published, the plugin line would look like:

```tmux
set -g @plugin 'startino/themes/tmux'
```

## Install manually

1. Copy the flavor conf you want into your tmux config directory:

```sh
cp dist/startino-neptune.conf ~/.tmux/startino-neptune.conf
```

2. Add this line to your `~/.tmux.conf` (or `tmux.conf`):

```tmux
source-file ~/.tmux/startino-neptune.conf
```

3. Reload tmux:

```sh
tmux source-file ~/.tmux.conf
```

## Switch flavors

Change the `source-file` line to point at a different conf:

| Flavor   | File                         | Tone  |
|----------|------------------------------|-------|
| Mercury  | `startino-mercury.conf`      | Light |
| Mars     | `startino-mars.conf`         | Dark  |
| Jupiter  | `startino-jupiter.conf`      | Dark  |
| Neptune  | `startino-neptune.conf`      | Dark  |

## Customisation note

Unlike Catppuccin tmux, there is no `@thm_status_module_*` accent rotation. The mint primary (`#45dfa4` on Neptune) is hardcoded as `@thm_primary` — this is a deliberate Startino design decision. The full palette is still exported as individual `@thm_*` variables, so you can reference any color in your own status-line modules.
